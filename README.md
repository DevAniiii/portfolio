<h1 align="center">Anirban's Portfolio</h1>

<p align="center">
  Modern cinematic developer portfolio built with Next.js, Tailwind CSS, Framer Motion, and immersive UI systems.
</p>

---

## ✨ Features

- Modern dark cinematic UI
- Fully responsive across all devices
- Smooth animations & immersive interactions
- Interactive project showcase
- Performance-focused architecture
- Glassmorphism & animated gradient systems
- Optimized Next.js application structure
- Dynamic sections with advanced UI effects

---

## 🚀 Technologies Used

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion

### UI & Animation
- Aceternity UI
- Lucide React
- Tabler Icons
- Custom Motion Systems

### Backend & Database
- Node.js
- MongoDB

### ⚡ Getting Started
## Clone Repository
```bash
git clone https://github.com/DevAniiii/portfolio.git
cd portfolio
```
### Install Dependencies
```bash
npm install
```
### Run Server
```bash
npm run dev
```
